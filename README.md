# RetailHub

Multi-Store Retail Management Platform

## Tech Stack

### Backend

- **Framework:** Quarkus 3.x
- **Database:** PostgreSQL 18
- **ORM:** Hibernate with Panache
- **Authentication:** JWT (SmallRye JWT)
- **Migration:** Flyway
- **Build:** Maven

### Frontend (Planned)

- **Framework:** Vue.js 3
- **Build:** Vite
- **State:** Pinia
- **Router:** Vue Router
- **HTTP:** Axios
